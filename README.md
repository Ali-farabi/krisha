# Квартиры в Астане 

Небольшой демо‑проект на Vue 3 + TypeScript + Vite. Мұнда мен пәтерлер тізімін шығарып, фильтр арқылы іздеу жасадым. Данные локальные, фильтры простые и понятные.

## не ыстедым 

- Фильтр по площади: **От / До**
- Фильтр по комнатам: **От / До**
- Поиск по адресу (подстрока, без регистра)
- Debounce 300ms на адрес
- Ошибка, если **От > До** (показывается подсказка)
- Показано: **Найдено N**
- Пустое состояние: **Ничего не найдено**
- Адаптив:
  - Desktop: фильтры слева, список справа
  - Mobile: фильтры сворачиваются (collapse)

## Стек / Технологии

- Vue 3 + TypeScript + Vite
- Element Plus (UI)
- Данные в `src/data/apartments.json`
