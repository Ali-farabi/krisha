# Фильтры квартир в Астане

Тестовое задание «Фильтры» на Vue 3 + TypeScript + Vite. Данные берутся из локального файла `src/data/apartments.json`. Интерфейс построен на Element Plus.

## Возможности

- Фильтрация по площади (от/до)
- Фильтрация по количеству комнат (от/до)
- Поиск по адресу (подстрока, без учета регистра)
- Debounce 300 мс для поля адреса
- Валидация диапазонов (если «От» больше «До», диапазон корректируется и показывается подсказка)
- Счетчик найденных квартир
- Пустое состояние «Ничего не найдено»
- Адаптивный layout: sidebar на desktop, stack на mobile

## Установка

```bash
npm i
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```
